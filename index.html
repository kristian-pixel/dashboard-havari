Hvad dækker kategorierne over?

Motorskade
Driftsstop, tekniske motorfejl, overophedning eller lignende problemer med fremdrift.
Grundstødning
Fartøjet er gået på grund, sidder fast eller har brug for assistance til at komme fri.
Ingen brændstof
Fartøjet mangler brændstof eller kan ikke fortsætte på grund af brændstofrelaterede problemer.
Garn i skrue
Line, garn, net eller andet er fanget i propel, drev eller ror.
Materiale skade
Skader på båd, udstyr eller installationer, som påvirker sejladsen.
Rorskade
Problemer med styring eller ror, hvor fartøjet ikke kan manøvrere sikkert.
Læk
Vandindtrængning eller mistanke om vandindtrængning.
SAR
Søredningsopgave, hvor der kan være tale om egentlig eftersøgning/redning eller alvorlig fare.
Andet / Ukendt
Hændelser der ikke passer rent i en anden kategori, eller hvor årsagen ikke er endeligt registreret.

Om Dansk Søredningsselskab og datagrundlaget

Dansk Søredningsselskab (DSRS) er en frivillig, non-profit søredningsorganisation med uddannede og ulønnede reddere på redningsstationer rundt i Danmark.

Data på denne side bygger på DSRS’ egne registreringer af assistancer. Opgørelserne er lavet til overblik, presseforespørgsler og samarbejdspartnere — ikke som en myndighedsstatistik.

Data opdateres løbende og kan ændre sig ved efterregistreringer, rettelser og kvalitetssikring.

Kreditering ved brug: Dansk Søredningsselskab (DSRS), www.dsrs.dk.

Om data og kilde

Brug af DSRS-data

Dansk Søredningsselskab (DSRS) er en frivillig, non-profit søredningsorganisation, der arbejder for at øge sikkerheden til søs i danske farvande. Arbejdet udføres af uddannede og ulønnede frivillige reddere.

Data på denne side er baseret på registrerede assistancer udført af DSRS. Tallene opdateres løbende og kan ændre sig ved efterregistreringer og kvalitetssikring.

Ved brug af data, grafer eller statistik fra siden skal Dansk Søredningsselskab krediteres som kilde.Kilde: Dansk Søredningsselskab (DSRS), www.dsrs.dk

Vigtigt: De fleste assistancer sker uden akut fare for liv. Ved akut fare på havet kontaktes altid 1-1-2.

Prototype – offentlig data er reduceret og anonymiseret til webvisning.
